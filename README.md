## About

Large language models and AI coding assistants are transforming how researchers write code for Earth Observation (EO) analysis — a practice increasingly known as "vibe coding." While these tools can dramatically accelerate workflows, they also introduce risks: hallucinated function calls, subtly incorrect geospatial parameters, unverified analytical pipelines, and a false sense of confidence in AI-generated outputs. This workshop equips conservation scientists and ecologists with a practical framework for the **responsible** use of AI-assisted coding in large-scale, rapid EO workflows. Participants will learn to harness AI tools effectively while maintaining scientific rigour, reproducibility, and transparency.
